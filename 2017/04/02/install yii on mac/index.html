在mac上安装yii

2017年4月2日

过去一直在windows环境开发，今年给自己添置了一台2015款的rmbp，以后就决定在mac环境开发了，后续的文章如无特殊说明，均为mac环境。
mac环境刚开始用还不太习惯，熟悉起来发现效率还是很高的。工作关系要使用Yii框架，本文算是一个安装入门篇。

安装Composer

因为天朝防火墙的原因，我们切换Composer的镜像为国内的

安装 Composer Asset Plugin

安装 Yii basic

因为引用了很多第三方资源，已经超出github的免费获取限制（貌似是这样），安装过程中需要输入一个github的token，登录你的github后，在settings里生成一个token即可。

稍等片刻安装完成，目录：/Users/youname/basic

进入安装目录，运行

正常情况下，你会看到Yii的页面了
